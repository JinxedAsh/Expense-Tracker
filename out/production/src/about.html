Personal Finance Tracker

This Personal Finance Tracker is a Java Swing application developed as a second-semester student project. It helps users manage their personal finances by tracking income, expenses, and bills.

Key Features:

  • Track income and expenses
  • Manage recurring bills
  • Set and monitor budgets
  • View financial summaries
  • Export data to CSV

Object-Oriented Programming Principles Demonstrated:

  • Encapsulation
  • Inheritance
  • Polymorphism
  • Abstraction
